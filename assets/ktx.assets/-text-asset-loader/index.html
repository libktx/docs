Allows reading text files with an AssetManager. Must be registered via setLoader.

Note that loadAsyncmust be called before loadSync, as usual in case of AsynchronousAssetLoader implementations. Similarly to standard libGDX loaders, this loader is not considered thread-safe and assumes that a single file is loaded at a time.

Parameters

fileResolver

not used, required by the superclass.

charset

name of the charset used to read text. Can be overridden with TextAssetLoader.TextAssetLoaderParameters. Should match text files encoding. Defaults to UTF-8.
